Короткий опис мого проєкту

Я створив працюючу лендінгову сторінку, створену за наданим макетом.
Вона містить головний блок який має своє лого та дає можливість зателефонувати та відкрити меню.
  Наводячи на іконки вони збільшуються, щоб показати що вони у працюючому стані.
Коли хтось нажимає на іконку меню, воно плавно виїжджає.
  Там ми бачимо, що маємо можливість переглянути секції такі як:
    Початок сторінки
    Інформація про велосипеди
    Порівнювання велосипедів
    Деталі
    І можливість зв'язатися
У мене є можливість через меню плавно переходити у потрібну секцію що супроводжуєть:
  При наведенні, під текстом виїждає тоненька лінія
  Після натискання, меню плавно закриваєть і Вас плавно перенаправляю на обрану секцію
Також моя сторінка адаптавона під комп'ютер, планшет та телефон


Технології, які було використано:
  HTML - семантична розмітка сторінки;
  SCSS - змінні, міксіни та вкладені стилі;
  GitHub Pages - платформа для деплою прев'ю;
  Figma - макет дизайну.
